 MeetingSMLR

Ad hoc meeting scheduler built using React, Node.js, MongoDB (MERN Stack), and WebSockets.

## Usage

```python
# input mongoDB credentials in server/nodemon.json
cd client
npm install
cd ../server
npm install
cd ..
npm start
```
